```txt
1.初始化
	N个节点，parent数组长度N。
2.查询
	每输入一对节点(u,v)，将其合并。
	//路径压缩和查询
	int find(int u){
    if(u != p[u]){
    		//都将其合并到最顶层节点
        p[u] = find(p[u]);
    }
    return p[u];
	}
3.合并
	调用find(u) == find(v)判断两节点是否连通。
	如果连通了，证明在一个连通块中，此时无需过多处理
	如果不连通，证明不在一个连通块中，此时可以合并，如果是带权值的，此时也要合并



获取集合数量：i == p[i]

关键是关系传递，在合并a和b的时候，其实是三个变量之间的关系传递，点a和p[a] 点b和p[b] p[a]和p[b]
比如：
	x = y 和 y = z 能得出x = z，有时候关系藏的比较深，要结合其它算法来看，比如结合前缀和等等

分类题型：
	普通并查集
	带权并查集（距离，统计，异或等等）
```
